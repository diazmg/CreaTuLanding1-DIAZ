React
Es una librería de JS para crear una interfaz de usuario (UI)
Por lo general se instala, y por lo general se usa para crear SPA (Single Page Application)

function SoyUnComponente() {

    return <p>Hola</p>
} // Esto, el navegador lo interpreta como error de sintaxis, ya que los signos <> no son compatibles con JS.
Por eso, tenemos que usar un compilador (Vite)


NodeJS --> Es un entorno de ejecución de JS fuera del navegador


ViteJS --> Es un entorno de ejecuón de JS (compilador)
Nos permite usar JS moderno en el navegador.
Se suele usar en la línea de comandos (gitBASH)

Para poder iniciar la ejeción de Vite, necesito tener instaldo NodeJS

Quiero instalar Vite, porque quiero usar React

npm create vite@latest .

npm install : para instalar las dependencias del proyecto basado en el package.json

npm run dev : para iniciar el proyecto en modo desarrollo, sería el nuevo Live Server, me da el localhost


node_modules : no se toca, tiene las cosas que se instalan (dependencias) (por ahora lo único fue vite)

public : sirve para guardar imagenes, tipografías, algún .cosas

src : la carpeta del proyecto como tal, es la carpeta más fuerte
